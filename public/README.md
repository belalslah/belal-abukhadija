# Public Assets

This folder contains static assets that are served directly by Next.js.

## Required Images

To complete the setup, add the following images:

### Favicon
- `favicon.ico` - 32x32 or 48x48 pixel icon

### Open Graph Image (for social media sharing)
- `og-image.png` - 1200x630 pixels
- This image appears when sharing on social media
- Should include your name and a visual representation of your brand

### PWA Icons (for Progressive Web App)
- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels
- These enable "Add to Home Screen" functionality

## Creating Images

### Tools You Can Use
- [Canva](https://canva.com) - Easy-to-use design tool
- [Figma](https://figma.com) - Professional design tool
- [Photoshop](https://adobe.com/photoshop) - Advanced image editing
- [Favicon Generator](https://favicon.io/) - Generate favicons

### Design Guidelines

**Favicon:**
- Simple, recognizable icon
- Good contrast for both light and dark backgrounds
- Consider using your initials or a logo

**Open Graph Image:**
- Include your name prominently
- Use your brand colors (light and green theme)
- Add a tagline or description
- Ensure text is readable at small sizes

**PWA Icons:**
- Should be the same design as favicon but larger
- Include some padding around the edges
- Ensure it looks good on various backgrounds

## Example Content

For testing purposes, you can use placeholder images initially:
- Use any square image for icons
- Use a 1200x630 banner image for og-image.png

Remember to replace these with your actual branding images before deploying to production!
